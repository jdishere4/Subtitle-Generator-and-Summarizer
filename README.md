# Subtitle Generator & Video Summarizer

## Project Overview
This project automates the process of generating subtitles from videos and summarizing the video content using Natural Language Processing (NLP) techniques.

## Objective
- Download video content
- Convert video to audio
- Generate text transcripts from audio
- Create subtitles from the transcript
- Generate a concise summary of the video content

## Tools & Technologies
- Python
- Speech Recognition
- NLP (Text Processing & Summarization)
- Jupyter Notebook

## Project Workflow
1. Video download
2. Audio extraction from video
3. Speech-to-text conversion
4. Subtitle generation
5. Text summarization

## Conclusion
This project demonstrates an end-to-end NLP pipeline for automating subtitle generation and content summarization, useful for video analysis and content consumption.
